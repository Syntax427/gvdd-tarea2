## Compilación y ejecución de la experimentación

Ejecuta los siguientes comandos dentro del directorio del proyecto:

```bash
g++ -o mrl main.cpp mrl_sketch.cpp
./mrl
```
El programa generará las tablas resultantes en formato `.csv`.

---

## Reproducción de las figuras

Para reproducir las figuras solo basta con ejecutar el script en Python:

```bash
python3 read.py
```
Esto generará las figuras en formato `.png`.
